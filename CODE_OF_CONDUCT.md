# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of
conduct.

The full text of the Contributor Covenant v2.1 is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

A translated Spanish version is available at:
<https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/>

## Summary

All participants in the HOC community — contributors, maintainers, reviewers,
issue reporters, and users — are expected to foster an open, welcoming,
diverse, inclusive, and healthy community. We commit to treating every
participant with respect and dignity regardless of age, body size, visible
or invisible disability, ethnicity, sex characteristics, gender identity
and expression, level of experience, education, socio-economic status,
nationality, personal appearance, race, caste, color, religion, or sexual
identity and orientation.

## Scope

This Code applies to all project spaces: GitHub repository (issues, pull
requests, discussions, code review comments), any official communication
channel, and when an individual is officially representing the project in
public spaces.

## Enforcement

If you observe or experience behavior that violates this Code of Conduct,
please report it to the project maintainers by email: **csrrtr@gmail.com**
with subject prefix `[HOC-CoC]`. All reports will be reviewed and
investigated promptly and fairly. The maintainers are obligated to respect
the privacy and security of the reporter.

Consequences for violations follow the **Enforcement Guidelines** from
Contributor Covenant v2.1: correction, warning, temporary ban, and
permanent ban, applied proportionally.
